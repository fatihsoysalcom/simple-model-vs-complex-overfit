# Simple Model vs Complex Overfit

This example demonstrates Occam's Razor in machine learning by comparing a simple linear regression model with a complex high-degree polynomial model on synthetic noisy data. It illustrates how a simpler model, despite not perfectly fitting the training data, can generalize better and be preferred over an overfitted complex model.

## Language

`python`

## How to Run

1. Ensure Python and pip are installed.
2. Install required libraries: `pip install numpy scikit-learn matplotlib`.
3. Run the script: `python main.py`.
4. A plot will be displayed showing the models' fits and a console output comparing their performance.
